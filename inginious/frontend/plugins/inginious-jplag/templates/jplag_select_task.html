Before any use

This plugin should allow administrator of a course to run JPlag tools to detect plagiarism. In case of problem, feel free to send an email to : Sysadmin INGI.

The server that performs the plagiarism check waits for a specific architecture:

foldertotest
->first_student
file_to_test
->second_student
other_file
->...

To get this structure, please add this little part of script to your run file.
How to:
Python way
from inginious import input
student_name = input.get_input("@username")
subprocess.call(['archive', '-a', 'student/FileIWantToTest.java','-o',student_name+''], universal_newlines=True)
subprocess.call(['archive', '-a', 'student/MyOtherFileIWantToTest.java','-o',student_name+''], universal_newlines=True)
This allows JPlag to find wich files must be tested.
